# Modularen-Daten-Ontologie-Plattform-MDOP-

Entwicklung einer Modularen Daten-Ontologie-Plattform (MDOP) zur Integration, Modellierung und Analyse heterogener Datenquellen. Ziel: Aufbau einer hoch skalierbaren, sicheren und nutzerzentrierten Datenplattform, die komplexe Beziehungen und Muster in Daten durch eine flexible, Graph-basierte Ontologie aufdeckt.

### Senior Compass App

An accessible, senior-friendly application module designed to help elderly users live independently while providing peace of mind to their families. This implementation includes:

- **Card Game Module**: A complete Python implementation of card game classes (Card, Deck, Player, Game)
- **Product Requirements Document**: Comprehensive PRD with UI wireframes, user flows, and backend architecture
- **Full Test Suite**: 21 unit tests with 100% pass rate
- **Accessibility-First Design**: Large buttons, high contrast, audio feedback

For detailed information, see [Senior Compass README](senior_compass/README.md).

#### Quick Start

```bash
# Run the card game demo
cd senior_compass/card_game
python demo.py

# Run tests
cd senior_compass/tests
python test_card_game.py
```
